Documentação:
Relatório-XXX.txt - Um total de 3 documentos contendo o relatório individual de cada um dos componentes do grupo, e no local de XXX nos documentos estará o nome do integrante a quem o relatório pertence.

Arquivos de código fonte que compõem o trabalho:
Grafo.c, Grafo.h - módulos de implementação e definição do trabalho, respectivamente, módulos que servem para criação de grafo
Vertice.c, Vertice.h - módulos (implementação e definição) que servem de apoio para os módulos Grafo.
TESTAGRAFO.c - é o módulo de implementação do módulo de teste específico do grafo.

Funções Do Módulo Grafo:

CRIAR_GRAFO - Cuja função é alocar na memória a estrutura grafo
Insere_No_Grafo - Serve para inserir informações no grafo
ObterValor - Com ela se localiza o valor de um vértice
ObterValorCorrente - Ela serve para identificar o valor em que o cursor se encotra 
IrInicioGrafo - Vai para o primeiro vértice inserido
IrFinalGrafo - Vai para o último inserido
Avanca_Corrente - Faz o cursor ir para o seguinte 
Elimina_No - Deleta o nó

testagrafo.script - (deveria)serve para o teste automamatizado do módulo Grafo
